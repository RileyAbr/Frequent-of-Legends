# Frequent-of-Legends
Research project into mining frequent patterns from League of Legends match data. Project composed of data collection, data storage, frequency mining, classification, and analysis. Over 20k matches worth of data was collected from the Riot API via Python and stored in a MongoDB database. Data was then used in mining algorithms in both Python and R.
